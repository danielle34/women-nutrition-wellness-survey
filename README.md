# Women’s Nutrition & Wellness Survey Demo

This repository contains a demo version of a **Women’s Nutrition & Wellness Intake Form**, built as a single-page HTML/CSS website for healthcare professionals. The form consolidates validated survey instruments including PSQ, PSQI, SF-12, FAS, EPDS, AHC, and core domains of the DHQ-II.

The purpose of this demo is to:
- Showcase how a unified wellness intake survey can be structured digitally  
- Demonstrate clean, accessible UI design for clinical environments  
- Support stakeholder review before full platform integration  

## Features
- Single-page responsive layout  
- Professional healthcare-friendly styling  
- Clean section navigation  
- Fully structured question grouping (Demographics, Health, Sleep, Mental Health, Diet)  
- Ready for client or practitioner review  

## How to Use
1. Open `index.html` in any browser  
2. Review the layout and question flow  
3. Customize the form as needed for clinical workflows
